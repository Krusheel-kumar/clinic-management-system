# User Stories - Smart Clinic Management System

## Doctor
- As a doctor, I want to view my appointments so that I can manage my schedule.
- As a doctor, I want to update my availability so patients can book slots.

## Patient
- As a patient, I want to register myself so I can book appointments.
- As a patient, I want to view available doctors.
- As a patient, I want to book an appointment with a doctor.

## Admin
- As an admin, I want to add doctors to the system.
- As an admin, I want to manage patient records.
- As an admin, I want to monitor appointments.
