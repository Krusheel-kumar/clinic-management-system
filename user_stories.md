# User Stories

## 👨‍⚕️ Doctor

### Story 1
- *Title:* Manage Appointments
- *As a:* Doctor
- *I want to:* View and manage my appointments
- *So that:* I can handle patient schedules efficiently
- *Acceptance Criteria:*
  - Doctor can view all appointments
  - Doctor can update appointment status


### Story 2
- *Title:* Update Availability
- *As a:* Doctor
- *I want to:* Set my available time slots
- *So that:* Patients can book appointments
- *Acceptance Criteria:*
  - Doctor can add time slots
  - Doctor can update availability




## 🧑‍🤝‍🧑 Patient

### Story 1
- *Title:* Register Account
- *As a:* Patient
- *I want to:* Register in the system
- *So that:* I can book appointments
- *Acceptance Criteria:*
  - Patient can sign up using email
  - Patient data is stored


### Story 2
- *Title:* Book Appointment
- *As a:* Patient
- *I want to:* Book an appointment with a doctor
- *So that:* I can get treatment
- *Acceptance Criteria:*
  - Patient can view doctors
  - Patient can select date & time



## 🛠 Admin

### Story 1
- *Title:* Add Doctor
- *As a:* Admin
- *I want to:* Add new doctors
- *So that:* Patients can consult them
- *Acceptance Criteria:*
  - Admin can add doctor details
  - Doctor is saved in database


### Story 2
- *Title:* Manage Patients
- *As a:* Admin
- *I want to:* View and manage patient records
- *So that:* I can monitor system usage
- *Acceptance Criteria:*
  - Admin can view all patients
  - Admin can delete/update patient data
